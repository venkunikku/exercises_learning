<td class="answercell">
    <div class="post-text" itemprop="text">
<p>Your Graph class takes no arguments on <code>__init__</code> therefore when you call:</p>

<blockquote>
  <p>graph = Graph(g)</p>
</blockquote>

<p>You get an error because Graph doesn't know what to do with 'g'.
I think what you may want is:</p>

<pre><code>class Graph(object):
    def __init__(self, values):
        self.__graph_dict = values
    def vertices(self):
        return list(self.__graph_dict.keys())
</code></pre>
